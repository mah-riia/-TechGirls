📋 Descrição do Projeto
O TechGirls CP é uma aplicação web desenvolvida como parte do Projeto 1 da disciplina de Programação Web Front-End da UTFPR, campus Cornélio Procópio.

O objetivo principal do projeto é mitigar a disparidade de gênero latente na área de tecnologia em nossa região, atuando diretamente em prol da comunidade local. Alinhado ao Objetivo de Desenvolvimento Sustentável 5 (ODS 5 - Igualdade de Gênero) da ONU, o website funciona como uma plataforma informativa e de engajamento para conectar mulheres e estudantes ao universo da computação, reunindo referências de capacitação gratuita e promovendo uma rede de apoio acolhedora.

A aplicação simula o fluxo completo de interação de uma usuária, contando com interfaces dedicadas de cadastro, autenticação e uma página principal com conteúdos educativos e informações sobre o ingresso nos cursos de tecnologia ofertados localmente pela UTFPR-CP.

🛠️ Tecnologias Utilizadas
Para construir a interface semântica e a estilização visual moderna da aplicação, foram utilizadas estritamente as tecnologias nativas solicitadas, sem o uso de frameworks externos:

HTML5: Estruturação semântica de layouts e componentes.

CSS3: Estilização baseada em variáveis CSS, layouts flexíveis (Flexbox e CSS Grid) e responsividade mobile.

Google Fonts & Boxicons / Material Icons: Tipografia externa integrada e iconografia para melhoria de acessibilidade.
